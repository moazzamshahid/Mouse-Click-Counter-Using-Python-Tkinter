# Mouse-Click-Counter-Using-Python-Tkinter

Configuration
1. Install python 3 from the link below.
Link: https://www.python.org/downloads/

Install Python 3.7 or 3.8.
2. Make Sure you have installed following python libraries
a. Pandas: Open command terminal from start and write command pip install pandas
b. Mouse: Pip Install mouse
c. Keyboard: pip install keyboard

Working:
1. Open the folder Clicker
2. Double click Counter.bat and GUI.bat file in Clicker folder.
3. Do not delete any files.
